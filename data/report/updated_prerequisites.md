# Prerequisite Relationships Report (Section-Level Only)

This document outlines the prerequisite relationships between **Sections (Middle Units: 중단원)** in the high school mathematics ontology. All concept-level links have been removed to ensure a high-level logical flow.

**Format**: `Prerequisite (Pre)` -> `Target (Post)`

## 1. Common Math 1 (공통수학1) & 2 (공통수학2)
### Polynomials & Equations
- 다항식의 연산 -> 인수분해
- 나머지정리 -> 인수분해
- 복소수 -> 이차방정식
- 인수분해 -> 이차방정식
- 이차방정식 -> 이차방정식과 이차함수
- 이차방정식 -> 여러 가지 방정식

### Sets, Functions, Data
- 집합의 뜻과 포함 관계 -> 집합의 연산
- 집합의 연산 -> 명제
- 집합의 연산 -> 함수
- 유리함수 -> 무리함수

## 2. Algebra (대수: 수원)
- 지수 -> 로그
- 지수 -> 지수함수
- 로그 -> 로그함수
- 삼각함수 -> 삼각함수의 그래프
- 삼각함수의 그래프 -> 삼각함수의 활용
- 등차수열과 등비수열 -> 수열의 합
- 등차수열과 등비수열 -> 수학적 귀납법

## 3. Calculus 1 (미적분1: 수투)
- 함수 -> 함수의 극한
- 함수의 극한 -> 함수의 연속
- 함수의 연속 -> 미분계수와 도함수
- 미분계수와 도함수 -> 도함수의 활용
- 도함수의 활용 -> 부정적분
- 부정적분 -> 정적분
- 정적분 -> 정적분의 활용

## 4. Probability & Statistics (확률과 통계)
### Internal Flow
- 경우의 수와 순열 -> 조합
- 순열과 조합 -> 이항정리
- 순열과 조합 -> 확률의 뜻과 활용
- 확률의 뜻과 활용 -> 조건부확률
- 확률의 뜻과 활용 -> 확률분포
- 확률분포 -> 통계적 추정

### Cross-Subject Connections
- **Sets (Common Math 2)**: 집합의 뜻과 포함 관계 -> 확률의 뜻과 활용
- **Counting (Common Math 1)**: 경우의 수와 순열 -> 순열과 조합
- **Calculus**: 정적분 -> 확률분포

## 5. Calculus 2 (미적분2: 미적분)
- 등차수열과 등비수열 -> 수열의 극한
- 수열의 극한 -> 급수
- 지수함수 -> 지수함수와 로그함수의 미분
- 로그함수 -> 지수함수와 로그함수의 미분
- 삼각함수 -> 삼각함수의 미분
- 미분계수와 도함수 -> 여러 가지 미분법
- 여러 가지 미분법 -> 도함수의 활용 (Transcendental)
- 정적분 -> 여러 가지 함수의 적분
- 여러 가지 미분법 -> 치환적분법과 부분적분법

## 6. Geometry (기하)
### Internal Flow
- 이차방정식 -> 이차곡선
- 이차곡선 -> 이차곡선의 접선
- 벡터의 연산 -> 벡터의 성분과 내적
- 공간도형 -> 공간좌표
- 공간좌표 -> 도형의 방정식

### Cross-Subject Connections
- **Coordinates (Common Math 2)**: 평면좌표 -> 공간좌표
- **Circles (Common Math 2)**: 원의 방정식 -> 공간좌표
- **Lines (Common Math 2)**: 직선의 방정식 -> 공간도형
