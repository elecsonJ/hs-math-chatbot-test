# 제안하는 추가 선수과목 연결 (Proposed Prerequisites)

선생님께서 말씀하신 "원의 방정식 -> 구의 방정식" 처럼, 더 촘촘한 연결을 위해 아래 관계들을 추가로 제안합니다.

### 기하 (도형의 확장)
- [ ] 원의 방정식 -> 구의 방정식 (2D -> 3D 확장)
- [ ] 직선의 방정식 -> 평면의 방정식 (1D -> 2D 확장)
- [ ] 점과 직선 사이의 거리 -> 점과 평면 사이의 거리
- [ ] 두 직선이 이루는 각 -> 두 평면이 이루는 각
- [ ] 삼수선 정리 -> 이면각 (공간지각 기초)

### 대수 & 해석 (함수의 확장)
- [ ] 다항식의 연산 -> 함수 (식 -> 함수의 기초)
- [ ] 이차방정식 -> 이차부등식 (방정식 해 -> 부등식 범위)
- [ ] 이차방정식 -> 이차함수의 최대, 최소 (대칭축 개념 활용)
- [ ] 지수 -> 지수함수의 그래프
- [ ] 로그 -> 로그함수의 그래프
- [ ] 삼각함수 -> 삼각함수의 그래프

### 미적분 (심화 연결)
- [ ] 수열의 극한 -> 급수의 수렴과 발산 (수열의 합의 극한)
- [ ] 함수의 극한 -> 지수함수와 로그함수의 극한
- [ ] 함수의 극한 -> 삼각함수의 극한
- [ ] 미분계수 -> 접선의 방정식 (기하학적 의미)
- [ ] 도함수 -> 속도와 가속도 (물리적 활용)
- [ ] 정적분 -> 속도와 거리

### 확률과 통계 (단원 간 유기적 연결 강화)
- [ ] 경우의 수 -> 확률의 뜻 (확률의 고전적 정의 = n(A)/n(S))
- [ ] 순열 -> 확률의 뜻
- [ ] 조합 -> 확률의 뜻
- [ ] 이항정리 -> 독립시행의 확률 (이항계수의 활용)
- [ ] 확률의 뜻 -> 확률변수와 확률분포 (확률 개념이 통계의 기초)
- [ ] 독립시행의 확률 -> 이항분포 (독립시행의 분포화)
- [ ] 이항분포 -> 정규분포 (큰 수의 법칙, 정규분포 근사)
- [ ] 확률변수와 확률분포 -> 이산확률변수의 기댓값과 표준편차
- [ ] 이산확률변수의 기댓값과 표준편차 -> 이항분포
- [ ] 연속확률변수의 확률분포 -> 정규분포
- [ ] 정규분포 -> 모평균의 추정 (표본평균의 분포)
- [ ] 정규분포 -> 모비율의 추정 (표본비율의 분포)
